Import and Export

OpenGNTG supports importing an object file to your canvas. Your shapes can also be exported from the canvas.

Import

Importing a shape or multiple shapes requires an object file stored as JSON file. To import the file, select File > Import in the toolbar, and then select the object file from your computer. The object will imported and placed in the canvas.

Export

Select the shapes you want to export in the object list. Click export in the propery bar to export the shape into you computer. The shape object will be stored as a JSON file.
