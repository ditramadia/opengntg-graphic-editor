Polygon

Polygon is another shape featured in OpenGNTG. In other graphic editing softwares, it is commonly referred as the pen tool. The polygon tool helps you create complex shapes.

Drawing the polygon

Select the polygon tool in the toolbar. Click the canvas to start drawing the polygon. Click on the canvas again to place another vertex, adding new vertex to your polygon. Close the polygon by clicking the first vertex to finish drawing the polygon.

Add new vertex

Select the polygon in the toolbar. Click Edit Shape in the property bar. Click at the canvas to place the new vertex. Click Done when finished.

Delete vertex

To delete a vertex, select the polygon in the toolbar. Click Edit Shape in the property bar. Click the vertex you want to delete in the canvas to delete them. Click Done when finished.
