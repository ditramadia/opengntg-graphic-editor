Editing Shapes

You can edit shape attributes in the property bar, located at the right of the page.

Selecting shapes and vertices

To edit a shape, select the shape by checking the check box in the object list, located at the left of the page. It also lets you select the individual vertices of the shape. You can also select multiple shapes and vertices to edit them all together.

Change shape and vertex color

Select the shape or vertex in the object list. Change the color in the property bar.

Move

Select the shape or vertex in the object list. Change the x and y coordinate.

Alternatively, you can drag and drop the shape or vertex directly in the canvas.

Dimension & size

Select the shape or vertex in the object list. Change the width or height in the property bar.

Rotate

Select the shape or vertex in the object list. Change the rotation property in the property bar.

Scale

Select the shape or vertex in the object list. Drag the vertex directly in the canvas to scale them while maintaining its aspect ratio.
