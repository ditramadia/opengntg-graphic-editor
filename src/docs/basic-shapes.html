Basic Shapes

Basic shapes in OpenGNTG includes line, square, and rectangle.

Line

To create a line, select the line tool. Draw the line in the canvas by clicking and dragging.

Square

To create a square, select the square tool. Draw the square in the canvas by clicking and dragging.

Rectangle

To create a rectangle, select the rectangle tool. Draw the rectangle in the canvas by clicking and dragging.
